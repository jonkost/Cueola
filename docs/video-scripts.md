# Training Video Scripts & Click-Paths (v2.1, 2026-07)

Ten videos: one per Learning Hub lesson plus the Instructor Quick Start.
**Voiceover scripts are NOT duplicated here** — the narration text for each
lesson lives in [content-reference.md](content-reference.md) (generated from
the in-app lessons; reading it aloud IS the script, and the Kokoro MP3s are a
timing reference). This file adds what the camera does: the click-path to
perform while each part of the narration plays.

Recording is an owner errand and may trail past Aug 2 (decision 14 decides
hosting; lessons already render a **Watch the video** slot when a lesson's
`video:` field gets a URL).

Conventions: record at 1280×800, cursor visible, one take per numbered beat,
pause a beat on every toast. Demo content: **Load Demo** unless stated.

---

## V1 · What The Three Tools Do — `LH-start` (~2 min)
1. Front page: hover the three cards (Cueola / Planda Bear / Flowmingo) as the
   intro names them.
2. **Load Demo** → rundown appears.
3. Topbar: open **Guide**, show the lesson list, click through two lessons.
4. Point at the session code badge while the narration hits "one shared
   session code".

## V2 · Your Profile And Portal — `LH-profile` (~2.5 min)
1. Front page → profile button (top corner) → **Your Cueola profile**.
2. **Create profile**: type a class login code, name, username; pick avatar +
   theme (wizard steps in order).
3. Sign out → **I have a username** → type it → straight in (no password —
   linger on that).
4. Portal: session card with position, to-do badge, unseen-notes badge;
   click **Open Cueola**, then back, then **Notes**.

## V3 · Build A Rundown — `LH-cueola-build` (~3 min)
1. **Blank Slate** → new code.
2. **Add Row** ×3; set one row **Timed**, one **Flex**.
3. Click cue cells: add a Video cue, an Audio cue, a Script cue.
4. **Edit** mode: drag-reorder a row, rename a label.
5. Dashboard (second window): open a finished session → **Next Episode**
   panel → **Start Next Episode →**; show the auto-name and ↳ From chip.

## V4 · Run Live Without Panic — `LH-cueola-live` (~3.5 min)
1. **Go Live** from demo; camera on the pre-live checks incl. **Playout
   first GO**.
2. Show the link strip (CLOUD · TALENT · PLAYOUT · SCRIPT) + caller badge.
3. Advance with arrows; click a row.
4. GO on a linked row → automatic call READY · TRACK · ROLL · TAKE; run it
   again and abort with **S**.
5. Toggle **Manual TAKE (armed call)**: GO arms, **TAKE · G** fires.
6. Paste a line into the question lane → Enter → talent shows the QUESTION
   card → Esc clears.
7. Kill the talent window → TALENT dark → **System status** → **Recover
   Flowmingo**.

## V5 · Prep The Show Package — `LH-plandabear` (~4 min)
1. Enter with a profile; open **Planda Bear**.
2. Call Sheet: fill three fields. Flip through Schedule / Safety Plan / patch
   sheets.
3. Production Notes: post, tag a department, reply in a thread, @-mention.
4. To-Do with assignee; checklist with per-item owners; show them landing on
   a portal (second window as the student).
5. Pin a note; show **Seen by**.
6. Group bar: **Reviewing** picker (instructor) and **Switch group**
   (student); **Pick your group** door.
7. **Export PDF Package** → preview → stamp.

## V6 · Set Up The Talent Display — `LH-flowmingo-talent` (~2.5 min)
Click-path follows the lesson steps: open Talent Display, enter code, READY,
local Controls once (speed/size/mirror), fullscreen, then hands off.

## V7 · Run The Remote Prompter — `LH-flowmingo-remote` (~3 min)
Remote Op beside a talent window: Play, brake/boost holds, size keys,
direction flip, Reset; end on sent-vs-applied status while killing the talent
window mid-command.

## V8 · Cue And Play Media — `LH-outrangutan` (~4 min)
1. Home card → **Session** mode.
2. Drop three files; trim one in the clip editor (waveform on audio).
3. Continue modes: Manual / Continue / Follow on three cues.
4. SFX Board: drop a sound, name it, hotkey it, fire it.
5. Output window to a display, **Identify**.
6. Link a rundown row to a cue; advance in Live → auto-fire.
7. **+ Learn a control** with a MIDI pad (or `Outrangutan.midiInject` in the
   console when no hardware).
8. **Cmd+S** → .ogshow; print the show pack.

## V9 · First Fixes — `LH-support` (~2.5 min)
Stage each symptom, then the fix: mismatched code; blank Flowmingo → push from
Script Op; sent-not-applied → reload talent; wrong row → Prev/Next; stale
paperwork → reopen PB; damaged rundown → **Settings ▸ File ▸ History** →
restore a cloud snapshot (show the confirm wording + recovery copy).

## V10 · Instructor Quick Start (~5 min)
Follow [INSTRUCTOR_QUICK_START.md](INSTRUCTOR_QUICK_START.md) top to bottom on
the dashboard: mint a Class Key → create a session with **Entry
Requirement** → paperwork preset → groups + lock → roster/portal tour →
**Start Next Episode** → term-end revoke. Script = read the Quick Start
sections aloud.
